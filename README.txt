
                 iOS SDK for Simplify Commerce


  What is it?
  ------------

  An iOS Framework and Bundle to the Simplify Commerce payments platform.   If you do not
  have an account sign up at https://www.simplify.com/commerce.


  Installation
  ------------

  For installation instructions, please see the Simplify iOS installation instructions:

  http://www.simplify.com/payments-web/docs/sdk/ios

  Using the SDK
  --------------

  For examples see https://www.simplify.com/commerce/docs/sdk/ios.

  Version
  -------

  This is version 1.2.0 of the SDK.  See CHANGES.txt for a list of updates to the SDK.
  For an up-to-date version check at https://www.simplify.com/commerce/docs/sdk/ios.
